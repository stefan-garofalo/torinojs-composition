Use this section for project-side domain source indexes when the repo defines durable routed domain surfaces.

Readable domain articles should live at the project-owned routes the repo defines, such as `/docs/cli` or `/docs/harness`. Do not create `wiki/domains`; domain source indexes belong here under `wiki/content/docs/project/domains`.
